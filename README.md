# BAIclaw

BAIclaw is a desktop AI assistant built on OpenClaw.

## Links

- Website: [https://b.ai](https://b.ai)
- Docs: [https://docs.b.ai/BAIclaw](https://docs.b.ai/BAIclaw)
- Repository: [https://github.com/BAI-labs/BAIclawX](https://github.com/BAI-labs/BAIclawX)
- Support: [support@b.ai](mailto:support@b.ai)

## Development

```bash
pnpm run init
pnpm dev
pnpm test
pnpm run build:vite
```
